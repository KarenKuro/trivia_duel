## Description

[Nest](https://github.com/nestjs/nest) framework TypeScript starter repository.

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start:api:dev

# production mode
$ npm run start:api
```

## Running the admin

```bash
# development
$ npm run start:admin:dev

# production mode
$ npm run start:admin
```
